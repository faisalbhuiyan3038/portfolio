## Project Overview

ShopHive is an e-commerce platform developed as part of an internship at Nerd Castle Ltd. The project served as a learning experience for modern web development technologies, specifically focusing on Angular and .NET Core Web API implementation. This single-developer project demonstrates fundamental e-commerce functionalities while incorporating industry-standard practices.

## Key Features

- **Product Management**
  - Comprehensive product listing with dynamic sorting capabilities
  - Sort products alphabetically
  - Sort products by price (ascending/descending)
  - Filter products by categories
  - Responsive product grid layout

- **Shopping Cart System**
  - Redis-based cart management
  - Persistent cart data across sessions
  - Cart access restricted to authenticated users
  - Real-time cart updates

- **User Authentication**
  - Secure login and signup functionality
  - JWT (JSON Web Token) based authentication
  - Protected routes for authenticated users
  - Session management

- **Checkout Process**
  - Detailed order summary
  - Total cost calculation
  - Clear pricing breakdown

- **Custom Design Elements**
  - Original logo design
  - Custom graphics and branding elements
  - Consistent visual identity throughout the platform

## Technical Implementation

### Frontend Architecture
- **Framework**: Angular
- **UI Framework**: Bootstrap
- **Key Features**:
  - Lazy loading implementation for improved performance
  - Responsive design using Bootstrap's grid system
  - Component-based architecture
  - State management for cart and user sessions

### Backend Architecture
- **Framework**: ASP.NET Core Web API
- **Database**: Microsoft SQL Server
- **Caching**: Redis for shopping cart management
- **Authentication**: JWT-based authentication system

### Integration Points
- RESTful API endpoints for product management
- Redis integration for cart data persistence
- SQL Server integration for product and user data
- JWT implementation for secure API communications

## Challenges and Solutions

### Challenge 1: Frontend-Backend Integration
- **Problem**: First-time implementation of full-stack integration between Angular and .NET Core
- **Solution**: Extensive documentation review and systematic debugging approach
- **Outcome**: Successfully established stable communication between frontend and backend

### Challenge 2: Cart Management
- **Problem**: Implementing persistent cart functionality with Redis
- **Solution**: Utilized Redis as a caching layer for cart data
- **Outcome**: Reliable cart management system with proper user session handling

### Challenge 3: Authentication Flow
- **Problem**: Implementing secure user authentication
- **Solution**: Implemented JWT-based authentication with proper token management
- **Outcome**: Secure and reliable user authentication system
